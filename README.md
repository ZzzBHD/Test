**GitHub上传代码的命令行**

* 把项目先git clone下来
* 在文件夹中进行修改
* 在文件夹中打开终端
* git init
* git add .
* git commit -m "commit"
* git push -u origin master
* 输入用户名和密码
* 完成上传
